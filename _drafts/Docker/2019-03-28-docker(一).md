## Docker
	Docker是一个平台，供开发者和系统管理员使用 container 来开发、部署和执行应用。 操作系统层面的虚拟化技术。


### Images
	一个image是一个可执行的包，包括了执行一个应用所需要的每一个，代码、运行时环境、库、环境变量和配置文件
	构建镜像，分层(多层)存储
	
### Containers
	一个container是一个image的运行实例
	容器存储层
